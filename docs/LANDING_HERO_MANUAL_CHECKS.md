# Landing Hero — Manual Checks

## 1. Logo bigger and centered
- [ ] Keepsy logo is visibly larger (~1.4–1.8x) than before
- [ ] Logo is centered horizontally and vertically inside the tablet
- [ ] No excessive empty space around the logo

## 2. Tablet looks glassy and premium
- [ ] Tablet has translucent background (frosted glass)
- [ ] Backdrop blur is visible
- [ ] Subtle gradient overlay aligned with page gradient
- [ ] Soft border and inner highlight
- [ ] No harsh drop shadow; diffused shadow only
- [ ] Rounded corners (~24–28px)
- [ ] Clean, calm aesthetic (no sparkle)

## 3. Cards never overlap hero text at any viewport width
- [ ] Resize from ~320px to 1920px: cards stay outside headline/subhead/CTA
- [ ] No card overlaps "Imagine it. Generate it. Cherish it."
- [ ] No card overlaps subtext or buttons
- [ ] Safe zone margin is visible as clear gap around text

## 4. Animations don’t cross safe zone
- [ ] Floating cards animate (parallax, gentle float)
- [ ] No card moves into the text area during animation
- [ ] Cursor-follow movement stays bounded

## 5. Mobile shows fewer cards and stays clean
- [ ] Below 1024px: max 5 floating cards visible
- [ ] Desktop: up to 9 cards
- [ ] No crowding or overlap on small screens
- [ ] Safe zone margin increases on mobile (40px)

## 6. Lighthouse / CLS not worse
- [ ] Run Lighthouse (Performance + Accessibility)
- [ ] CLS (Cumulative Layout Shift) has not increased
- [ ] No new layout jumps on load
